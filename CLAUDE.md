## Project Overview

Herbivore is a Godot 4.6 game project using C# and the Mobile renderer.

## Development Commands

```bash
# Open project in Godot editor
godot --editor project.godot

# Run the game
godot project.godot

# Build C# solution (after adding C# scripts)
dotnet build
```

## Technical Configuration

- **Engine**: Godot 4.6 with Mobile rendering method
- **Language**: C# (.NET)
- **Physics**: Jolt Physics (3D)
- **Target**: Mobile platforms
